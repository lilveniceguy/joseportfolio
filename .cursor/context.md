This is a custom WordPress theme.

Tech stack:
- WordPress (PHP templates)
- Tailwind CSS v3 (compiled locally to dist/tailwind.css)
- Alpine.js v3 (copied locally to dist/alpine.min.js)
- No jQuery
- No CDN assets
- Layout is 100vh, no page scroll
- Sections switch via Alpine state
- Tailwind config uses default colors + optional tokens
